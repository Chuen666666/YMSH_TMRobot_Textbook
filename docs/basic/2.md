## 按鈕功能

手臂靠近末端模組的地方有許多按鈕，但因為某些比賽會將部分按鈕鎖起來，故此處僅會提及最常被使用的兩顆按鈕，至於其他按鈕（`SET` 和 `VISION`），你們就自己按按看就知道是什麼功能以及能不能用了。

![手臂上的按鈕](/button_on_eye.jpg)

- `POINT` 按鈕：按下後可以在專案中設置一個 `Point`（後面章節會詳細說明）
- `FREE` 按鈕：只有在按住 `FREE` 按鈕時，才能直接用手拉動手臂
