# 3. 收納重點

## 透明箱子

- Landmark 塊收到袋中
- 麻將收到袋中
- 放麻將的小板子放到袋中
- 校正板疊在上層
- 蓋子確定蓋上

## 線 & 外接設備

- 箱子放回原位
- 大屏推回原位
- HDMI 線拔掉並放回原位
- 電源線拔掉，卷好，並放回手臂外殼內
- 控制器放回原位（吸到鐵製控制器架上或鐵製區域）

## 手臂（正規使用姿態）

1. 點擊左上角的 `≡`，再點擊 `配置` &rarr; `姿態設定`，點選 `正規使用姿態`
2. 按住控制器 `PLAY`，直到調整完畢
3. 再次點擊左上角 `≡`，再點擊 `關機`，出現警告視窗後，點擊 `確定` 即可正常關機
