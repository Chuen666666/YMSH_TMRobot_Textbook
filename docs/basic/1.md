## 按鈕們

### 大紅按鈕

以舊版控制器的配置來說，在六顆小按鈕的右側，有一個大大紅紅的按鈕，即所謂的「緊急開關」。

正常情況下不需要動到它，一旦按下它，機械手臂會直接斷電停止，在 99.99% 的情況下都不會用到它，但萬一不小心按到了（例如控制器掉到地上，把按鈕拍下去了），那可以把那顆紅色按鈕順時針轉到底後，方可向上拉出，解除急停狀態。

### 六顆小按鈕

在緊急開關的右邊有著六顆功能按鈕，讓我們由上至下、由左至右，一一來認識它們：

- <img src="/power.png" alt="POWER 鍵圖示" style="width: 30px; display: inline; vertical-align: middle; margin: 0 5px;"> `POWER` 電源：開機機械手臂時使用
- <img src="/stop.png" alt="STOP 鍵圖示" style="width: 30px; display: inline; vertical-align: middle; margin: 0 5px;"> `STOP` 停止：停下執行中的專案或動作，或退出某些頁面之用
- <img src="/ma.png" alt="M/A 鍵圖示" style="width: 30px; display: inline; vertical-align: middle; margin: 0 5px;"> `M/A` 手動／自動切換：開機後須切換成手動模式始可操作
- <img src="/play.png" alt="PLAY/PAUSE 鍵圖示" style="width: 30px; display: inline; vertical-align: middle; margin: 0 5px;"> `PLAY/PAUSE` 執行／暫停：按下它來開始或暫停專案，或在部分操作時須依提示按住
- <img src="/plus.png" alt="+ 鍵圖示" style="width: 30px; display: inline; vertical-align: middle; margin: 0 5px;"> `+` 加：調整坐標時微調數值之用，或調整專案執行速度，或在部分操作時須依提示按住
- <img src="/minus.png" alt="- 鍵圖示" style="width: 30px; display: inline; vertical-align: middle; margin: 0 5px;"> `-` 減：調整坐標時微調數值之用，或調整專案執行速度
