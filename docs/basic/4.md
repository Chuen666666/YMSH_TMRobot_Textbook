# 4. 登入並取得權限

## 開機

在連接好電源和螢幕後，點擊控制器上的 `POWER` 即可開機。

## 取得權限

1. 點擊左上角的 `≡`，再點擊 <img src="/people_icon.jpg" alt="登入圖標" style="width: 25px; display: inline; vertical-align: middle; margin: 0 5px;"> 開啟登入視窗（新版界面會自動在開機後跳出，此步可略過）
2. 以 **Administrator** 權限登入，預設無密碼，直接點擊 `確定` 登入即可
3. 點擊 `取得控制權` 後（新版界面預設擁有控制權，此步可略過），再點擊控制器上的 `M/A` 以切換成手動模式（手臂上燈號為藍色表示自動、綠色表示手動）
4. 點擊 `≡`，再點擊 <img src="/TM_project_icon.png" alt="專案圖標" style="width: 25px; display: inline; vertical-align: middle; margin: 0 5px;"> 進到專案頁，創建新專案後，始可開始編寫專案（一律使用 `TMFlow`）
