# 2. 控制器上的紅色大按鈕

## 急停

在[前面章節](/basic/1)有提到過這顆緊急開關的使用方式以及解除方式，故本章將不再贅述。

## 開機防撞機

### 使用時機

手臂開機時，會產生小幅度的抖動，若此時它與周圍物品太近，便有可能導致撞機。

### 使用方式

在認為手臂開機時可能會撞機時，可以先按下控制器上的緊急開關再開機，此時手臂便會進入安全開機模式，將所有關節都鎖死後才進入系統。

進入系統時，它應會呈現類似報錯狀態，此時可以按住手臂上 `FREE` 按鈕，移動手臂（因關節處鎖定狀態，手臂會便得非常重，請放心操作），讓手臂遠離可能致使撞機之物後，再解除關節鎖。 <!-- TODO -->
