# 3. 初賽經驗談

::: warning 注意
此章所有內容皆以我們參賽經驗撰寫，僅供參考，一切規則以主辦單位公佈為準。
:::

## 流程和限制

一般來說，初賽要比一天，不過其實整個時間會被切成上午和下午。上午基本上就都是在教手臂的操作方式，一定會有許多已經學過的部分，和一點點沒有注意過的部分，建議可以聽一下，它和下午的比賽一定會很有關聯；下午則會有一點自主練習的時間，這段時間就把準確度和速度的策略和你的隊友練好、討論好，讓比賽時能夠發揮出最佳實力和最好的準確度。

初賽一般來說題目都蠻簡單的，主要是測速度，準確度當然也是一定要的，不過初賽允許的容錯稍大，只要按照課程教的加上平時練習的，要打敗其他學校應該就不難了，唯一的難點通常反而是受制於「一校最多進兩組」的限制而敗給自己的同學。

## 準備

比賽的工作分配基本上都會是一人負責手臂移動和控制器操作、一人負責操作軟體，所以在賽前，控制硬體的人應該練到可以較為精準地操作手臂，不過因為手臂實操作起來也真的不容易，所以記得在適當的時候使用控制器 `+` 或 `-` 進行微調（這也需要軟體操作者的配合）；操作軟體的人則要把軟體練到非常熟，最好是連畫面都還沒跳出來就能知道接下來應該怎麼操作。不過最重要的是，兩人應該常常一起練習，可以透過練習本教材的自主練習章節，並在過程中熟悉配合，甚至一些簡短的用語，以利高效溝通。

另外，其實可以在賽前先向達明（或找我們學校老師）詢問賽程表，從課程名稱中就能夠大致推斷出初賽的題目。（技巧來自 2025 手臂冠軍，~~真是的我當初怎麼沒想到這招~~）
